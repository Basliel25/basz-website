YAML-defined tmux project layouts. `mux <name>` to start.

## Projects

- `website` — frontend/backend split
- `MapReduce` — distributed systems sandbox
- `log_parser` — log analysis layout

## File pattern

`~/.config/tmuxinator/<name>.yml`

## Skeleton

```yaml
name: project
root: ~/path
windows:
  - editor:
      layout: main-vertical
      panes:
        - nvim
        - <run-command>
```
