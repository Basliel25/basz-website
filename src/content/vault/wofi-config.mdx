Wayland-native dmenu/launcher. Used for apps, clipboard, theme picker.

## Files

- `wofi/.config/wofi/config` — behavior
- `wofi/.config/wofi/style.css` — themed via [[Theme Engine]]

## Modes in use

- `--show drun` — app launcher (`SUPER + SPACE`)
- `--dmenu` — clipboard (`SUPER + V`), theme switcher (`SUPER + U`)
