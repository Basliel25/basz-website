Notification daemon + control center. Exposes count to [[Waybar Config]] via `custom/notification`.

## Files
- `swaync/.config/swaync/config.json`
- `swaync/.config/swaync/style.css` — theme-synced

## Spotify integration
Track-change notifications via `playerctl` watcher → `notify-send`. Uses album-art image path. See [[Custom Scripts]].

## Reload
`swaync-client -R`
