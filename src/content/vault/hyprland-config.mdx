Wayland tiling compositor. Modular config split by concern.

## Files

- `hyprland.conf` — entry point, autostart, monitor/workspace rules
- `keybinds.conf` — all bindings, sourced from main
- `plugins.conf` — `hyprexpo` and others
- `theme_0-53.conf` — generated by [[Theme Switcher Script]]; do not edit directly

## Monitor topology

```ini
monitor=HDMI-A-1,1920x1080@60,auto,1
monitor=eDP-1,preferred,auto,1
```

Workspaces 1–5 → `HDMI-A-1` · 6–10 → `eDP-1`. Pinned by ID, not by index.
