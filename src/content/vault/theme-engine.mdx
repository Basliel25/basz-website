## Animation Bezier Curves
The system utilizes high-precision cubic-bezier curves for organic movement:
- **`easeOutQuint`**: `0.23, 1, 0.32, 1`
- **`easeInOutQuart`**: `0.76, 0, 0.24, 1`

## Variable contract
Every `theme.conf` must export:
```
ACCENT1, ACCENT2
GAPS_IN, GAPS_OUT
HYPR_ROUNDING
OPACITY_TERMINAL
```
